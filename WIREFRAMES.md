# Dashboard Wireframes

**Version**: 1.0.0
**Last Updated**: 2026-01-24
**Purpose**: ASCII art wireframes for three-tier dashboard system

---

## Overview

This document contains wireframes for:
1. **Main Dashboard** - Overview of all bots
2. **Bot Dashboard** - Single bot with multiple strategies
3. **Strategy View** - Live workflow execution for one strategy

**Design Principles:**
- Unified UX across all tiers
- Live data on nodes (not static like n8n)
- Unity-style runtime editing with save buttons
- Clear visual hierarchy
- Information density optimized for desktop

---

## Animation Guidelines

### Node Execution Animation

**Active State (Executing):**
- Border: 2px solid green with pulse animation
- Duration: 300ms ease-in-out
- Scale: 1.02 (subtle grow)

**Success State:**
- Flash green background (200ms)
- Checkmark icon appears briefly

**Error State:**
- Border turns red
- Shake animation (300ms)
- Error icon persists

### Value Change Animation

**When value updates:**
- Number fades out (100ms)
- New number fades in (100ms)
- Highlight background briefly (yellow â†’ transparent, 500ms)

### Chart Animations

**Line chart updates:**
- New data point slides in from right
- Line smoothly extends (300ms ease-out)
- No jarring jumps
